    Чтобы установить нашу программу следуйте данным пунктам!

1. Убедитесь что у Вас установлен Python (версии 3.6.4 или новее)
2. В терминале проследуйте в деректорию installation
3. Проверьте работу pip и наличие файла requirements.txt
4. Введите команду:
        pip install -r requirements.txt
5. Запустите main.py
6. Радуйтесь!)
